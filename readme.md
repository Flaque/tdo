# todo [![Build Status](https://travis-ci.org/Flaque/todo.svg?branch=master)](https://travis-ci.org/Flaque/todo)

> My rad module


## Install

```
$ npm install --global todo
```


## Usage

```js
$ todo --help

  Usage
    todo [input]

  Options
    --name  Lorem ipsum [Default: false]

  Examples
    $ todo
    I love Ink
    $ todo --name=ponies
    I love ponies
```
